============
Installation
============


Stable release
--------------

To install chess_clock, run this command in your terminal:

.. code-block:: console

    $ pip install chess_clock

This is the preferred method to install chess_clock, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for chess_clock can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/mush42/chess_clock

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/mush42/chess_clock/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/mush42/chess_clock
.. _tarball: https://github.com/mush42/chess_clock/tarball/master
